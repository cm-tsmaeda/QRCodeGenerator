## 使用したJSライブラリ

こちらのライブラリを使用させていただきました。ありがとうございます。

[qrcodejs](https://github.com/davidshimjs/qrcodejs)

## 使い方

zipファイルをダウンロードして、解凍してください。<br>
index.htmlをブラウザで開くとQRコード生成画面が表示されます。
